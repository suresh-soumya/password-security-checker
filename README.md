# Advanced Password Security Analyzer

A Python-based cybersecurity tool that analyzes password strength, calculates entropy,
estimates brute-force crack time, and checks exposure in known data breaches using
secure hashing and k-anonymity.

## Features
- Secure password input masking
- Password strength evaluation
- Entropy calculation
- Time-to-crack estimation
- Breach detection (Have I Been Pwned API)
- Secure forensic logging (hashed passwords only)

## Technologies
- Python
- Cryptography (hashing)
- API security
- Secure coding practices

## How to Run
pip install -r requirements.txt  
python checker.py
